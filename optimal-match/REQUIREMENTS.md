# 最適人探し 要件定義 v1.1

## 概要

興味・目的・価値観・誠実さの4軸で相性スコアを算出し、最適なマッチ相手をランキング表示するマッチングアプリ。

## MVP機能

- [x] **アカウント登録・ログイン**（メール + パスワード、セッションCookie）
- [x] プロフィール登録（ユーザーごとに1件）
- [x] サンプルユーザー8名のシードデータ
- [x] 相性スコア算出（4軸）
- [x] 最適マッチ一覧（スコア順 + スタイルフィルター）
- [x] マッチ詳細（内訳・会話きっかけ）
- [x] 遊び型・誠実型判定

## 技術スタック

- Next.js 16 + TypeScript + Tailwind CSS
- SQLite (better-sqlite3)
- 認証: scrypt + セッショントークン（httpOnly Cookie）
