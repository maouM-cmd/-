# 最適人探し 要件定義 v1.0

## 概要

興味・目的・価値観の3軸で相性スコアを算出し、最適なマッチ相手をランキング表示するマッチングアプリ。

## MVP機能

- [x] プロフィール登録（名前・年齢・自己紹介・興味・探している関係・価値観）
- [x] サンプルユーザー8名のシードデータ
- [x] 相性スコア算出（興味40% + 目的30% + 価値観30%）
- [x] 最適マッチ一覧（スコア順ランキング）
- [x] マッチ詳細（内訳・理由表示）

## 技術スタック

- Next.js 16 + TypeScript + Tailwind CSS
- SQLite (better-sqlite3)
