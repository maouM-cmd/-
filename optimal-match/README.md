# 最適人探し

相性スコアで、あなたに最適な人を見つけるマッチングアプリ MVP。

## 機能

- **プロフィール登録** — 興味・目的・価値観（1〜5）
- **相性スコア** — 興味40% + 目的30% + 価値観30%
- **最適マッチ一覧** — スコア順ランキング
- **詳細内訳** — なぜ相性が良いかを表示

## セットアップ

```bash
cd optimal-match
npm install
npm run dev
```

http://localhost:3000

## 使い方

1. `/profile` でプロフィールを作成
2. `/discover` で最適マッチを確認
3. 気になる人をタップして相性内訳を見る
