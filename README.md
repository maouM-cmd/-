# T.N 恋愛シミュレーション

黒髪ロングでツンデレな **T.N** との会話選択で、
- 好感度
- 恋愛解説ポイント（T.Nからの指摘）
- T.N温度（会話履歴で変化）

を上げていく恋愛シミュレーションゲームです。

## コア体験（改良版）
- 2Dキャラ表示（黒髪ロング寄りの見た目、表情差分・衣装差分）
- **表情/衣装カスタム**：笑顔・照れ・怒り・驚き・眠い + 水色/茶色/黒服（2D切替）
- **恋愛解説の診断化**：各選択に「なぜ刺さったか」を表示
- **タイプ診断**：ツンデレ耐性 / 共感力 / 押し引きバランス を数値化
- **条件解放イベント**：行動傾向で特別シーンが開く
- **履歴連動の人格温度**：同じシーンでも反応温度が変化
- **ネガポジ判定カスタム**：ポジ感度・ネガ厳しさ・判定モードをユーザーが調整可能

## 追加したパターン
- 通常シーンを増量（教室 / 交差点 / 駅前 / 図書室 / 夜メッセージ）
- 各シーンの会話選択肢を拡張し、プレイごとの差分を増やした
- 条件次第で隠しイベント（屋上 / 夏祭り）が発生

## 遊び方
1. `index.html` をブラウザで開く
2. まずネガポジ判定の好み（厳しめ / 恋愛重視など）を設定
3. 各シーンで会話を選択して診断・温度・解放イベントを育てる
4. 最終エンドと診断を確認し、判定設定や選択傾向を変えて再挑戦する

## ファイル構成
- `index.html`: 画面レイアウト / 2DキャラUI
- `style.css`: UIスタイル（判定設定パネル・診断パネル・解放イベントパネル）
- `script.js`: ゲームロジック（診断、解放、温度、ネガポジ判定、分岐、2Dキャラ切替）

## メモ
- キャラ表示は2D DOM描画なのでオフラインでも表示できます。
