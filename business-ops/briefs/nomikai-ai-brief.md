# 飲み会盛り上げAI — プロジェクトブリーフ

| 項目 | 記入 |
|------|------|
| プロジェクト名 | nomikai-ai |
| 一言説明 | 幹事がリンクを共有するだけで、全員の予定・中間地点・店・盛り上げプランがスマホで見られる飲み会調整アプリ |

## 目的

- 飲み会の幹事が LINE 等で URL を共有し、参加者がログインなしで予定・最寄駅を入力
- 全員の中間地点・店舗候補・盛り上げプランを自動生成
- スマホから閲覧・コピーできる

## MVP機能

1. イベント作成 & 参加者入力（共有リンク方式）
2. 中間地点 & 店舗候補（HeartRails API + ルールベース）
3. 盛り上げ AI プラン（ルールベース、コピー UI）

## 技術選定

| 項目 | 選択 |
|------|------|
| フレームワーク | Next.js 16 + TS + Tailwind |
| DB | SQLite (better-sqlite3) |
| 参考プロジェクト | coupon-board, optimal-match |

## スコープ外

- ログイン・認証
- LLM / Google Maps API（Phase 2）
- プッシュ通知
