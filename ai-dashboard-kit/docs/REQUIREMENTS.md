# AI企業ダッシュボード — 要件定義

## 目的

主要AI企業の動向（企業情報・モデル・資金調達・ニュース）を一覧・比較できるダッシュボードを構築する。

## ユーザー

- AI業界のトレンドを追いたい開発者・投資家・研究者
- 複数社のモデル性能・価格を横断比較したい人

## 画面構成

| 画面 | パス（例） | 説明 |
|------|-----------|------|
| 概要ダッシュボード | `/` | KPIカード、資金調達チャート、最新ニュース |
| 企業一覧 | `/companies` | カード/テーブル、カテゴリ・タグフィルター |
| 企業詳細 | `/companies/[id]` | プロフィール、モデル、調達履歴、競合 |
| モデル比較 | `/models` | 価格・ベンチマーク・コンテキスト比較表 |
| 資金調達 | `/funding` | タイムライン、ラウンド一覧 |

## KPI（概要画面）

1. **追跡企業数** — `companies.length`
2. **合計評価額** — `valuationUsdB` の合計（null除外）
3. **累計調達額** — `fundingRounds.amountUsdM` の合計
4. **今年のモデルリリース数** — 当年の `models.releaseDate` 件数
5. **平均コンテキスト長** — `contextWindowK` の平均（null除外）

## フィルター・ソート

- 企業: カテゴリ、タグ、本社地域
- モデル: 会社、オープンウェイト有無、API有無、モダリティ
- 資金調達: ステージ、期間、リード投資家
- ソート: 評価額、従業員数、設立年、リリース日

## 非機能要件

- **データソース**: 初期は `ai-dashboard-kit/data/*.json` を静的インポート
- **言語**: UIは日本語メイン、企業名は日英併記
- **レスポンシブ**: モバイル〜デスクトップ
- **ダークモード**: 推奨（AI系ダッシュボードの慣習）
- **アクセシビリティ**: チャートにはテーブル代替を用意

## 技術スタック（推奨）

| 層 | 推奨 |
|----|------|
| フレームワーク | Next.js 15+ (App Router) |
| 言語 | TypeScript |
| スタイル | Tailwind CSS |
| チャート | Recharts または Chart.js |
| データ | 静的JSON → 将来はAPI/DB |

## データ契約

型定義は `schemas/types.ts` を正とする。JSONファイルはこの型に準拠すること。
