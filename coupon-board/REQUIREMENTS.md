# 招待みんなでショータイム 要件定義 v1.2

## 概要

ロケットなうのような**紹介・招待キャンペーン**を集めた掲示板。

## 確定事項

| 項目 | 内容 |
|------|------|
| サイト名 | **招待みんなでショータイム** |
| 特典 | 紹介者・被紹介者の**両方** |
| 投稿 | 誰でも自分の招待リンク/コードを**即時公開** |
| 編集・削除 | ユーザー側は不可（管理画面から削除可） |
| 同一サービス | **複数投稿OK** |
| スクリーンショット | **推奨**（任意添付） |
| 期限切れ案件 | **一覧から自動非表示** |
| 通報 | 3件で自動非表示、管理画面で復活・削除可 |
| 収益化 | **広告あり**（AdSense 対応枠） |
| 法的ページ | **利用規約**（/terms）・**プライバシーポリシー**（/privacy） |

## MVP機能

- 案件一覧（新着・人気・特典順、期限切れ自動除外）
- カテゴリ・検索
- 投稿（双方の特典 + リンク/コード + スクショ推奨）
- 詳細（特典、スクショ、リンク、コードコピー、役に立った）
- **コメント**
- **使えた / 使えなかった 報告**
- **お気に入り**（ブラウザ保存）
- **管理画面**（/admin、パスワード認証）
- **広告枠**

## 管理画面

- パスワード: 環境変数 `ADMIN_PASSWORD`（デフォルト: `shotime-admin`）
- 全案件の表示・非表示・削除
- 通報一覧の確認

## 広告設定

`.env.local` に以下を設定:

```
NEXT_PUBLIC_ADSENSE_CLIENT_ID=ca-pub-xxxxxxxx
NEXT_PUBLIC_ADSENSE_SLOT=xxxxxxxx
ADMIN_PASSWORD=your-secure-password
```

## スコープ外（Phase 2）

- 会員登録・ログイン
- ユーザーによる投稿編集
- プッシュ通知
