# 招待みんなでショータイム 要件定義 v1.1

## 概要

ロケットなうのような**紹介・招待キャンペーン**を集めた掲示板。紹介者・被紹介者の両方の特典を掲載し、誰でも自分の招待リンクを投稿できる。

## 確定事項

| 項目 | 内容 |
|------|------|
| サイト名 | **招待みんなでショータイム** |
| 特典の対象 | 紹介者・被紹介者の**両方** |
| 投稿内容 | ユーザーが**自分の招待リンク/コード**を載せられる |
| 投稿権限 | **誰でも**即時公開（ログイン不要） |
| 編集・削除 | MVPでは**不可** |
| 同一サービス | **複数投稿OK**（各自の招待リンク） |
| スクリーンショット | **任意添付**（JPEG/PNG/WebP、5MB以下） |
| 通報 | **あり**（3件で自動非表示） |

## MVP機能

- 案件一覧（新着・人気・紹介者特典順・被紹介者特典順）
- カテゴリ絞り込み・キーワード検索
- 案件投稿（双方の特典 + 招待リンク/コード + スクショ）
- 案件詳細（特典表示、スクショ、リンクを開く、コードをコピー、役に立った、通報）
- スマホ優先UI

## 通報理由

- 期限切れ・終了済み
- 虚偽・誤った情報
- 詐欺・違法の疑い
- スパム・重複
- その他

## スコープ外（Phase 2）

- 会員登録・ログイン
- 投稿の編集・削除
- 管理画面
- コメント・お気に入り
