# 初回クーポン掲示板

各サービスの初回クーポン・初回特典情報を共有できる掲示板サイトです。

## 機能

- **クーポン一覧** — 新着順・人気順で閲覧
- **検索・フィルター** — キーワード検索、カテゴリ別絞り込み
- **クーポン投稿** — サービス名、割引内容、クーポンコード、URL などを投稿
- **詳細ページ** — クーポンの詳細確認、外部リンク、役に立ったボタン
- **サンプルデータ** — 初回起動時にサンプルクーポンを自動投入

## 技術スタック

- [Next.js](https://nextjs.org/) 16 (App Router)
- TypeScript
- Tailwind CSS
- SQLite (better-sqlite3)

## セットアップ

```bash
cd coupon-board
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 本番ビルド

```bash
npm run build
npm start
```

## ディレクトリ構成

```
src/
├── app/              # ページ・API ルート
├── components/       # UI コンポーネント
└── lib/              # DB・型定義・定数
data/                 # SQLite データベース（自動生成）
```

## カテゴリ

| カテゴリ | 説明 |
|---------|------|
| EC・通販 | ネットショッピング |
| 食品・グルメ | デリバリー、飲食 |
| 美容・コスメ | コスメ、美容サービス |
| サービス | 各種サービス |
| サブスク | サブスクリプション |
| その他 | 上記以外 |
