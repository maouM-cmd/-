# 面接原稿カンニング

就活・転職の面接対策用 Web アプリです。質問ごとに回答の「要点」と「全文」を登録し、練習モードとクイック参照で本番に備えられます。

## 機能

- **就活 / 転職テンプレート** — 頻出質問を初期セットとして作成
- **企業別セット管理** — A社用・B社用など複数セットを保存
- **原稿編集** — 質問・要点・全文・目安時間を編集
- **練習モード** — タイマー付きで1問ずつ練習（要点 / 全文の切り替え）
- **クイック参照** — 大きな文字で要点を一覧 or 1問表示（面接直前の確認用）
- **データ保存** — ブラウザの localStorage（JSON エクスポート / インポート対応）

## セットアップ

```bash
cd interview-cue
npm install
npm run dev
```

http://localhost:3000 を開いてください。

## 使い方

1. ホームで「就活」または「転職」テンプレートからセットを作成（企業名を入力）
2. 質問一覧から各質問の「編集」で要点・全文を入力
3. **練習モード** — 自宅で声に出して反復練習
4. **クイック参照** — 面接直前に要点をさっと確認

## 技術スタック

- Next.js 16 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- localStorage（サーバー不要）

## スクリプト

| コマンド | 説明 |
|---------|------|
| `npm run dev` | 開発サーバー起動 |
| `npm run build` | 本番ビルド |
| `npm run start` | 本番サーバー起動 |
| `npm run lint` | ESLint |

## データについて

データはブラウザの localStorage に保存されます。キャッシュ削除や機種変更時は、ホーム画面の「JSON エクスポート」でバックアップを取ってください。
